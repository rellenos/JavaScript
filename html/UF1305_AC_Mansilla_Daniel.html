Ejercicio 4

1 Operacions bàsiques, Tipus de dades, Prompt i Alert

1.1 Crear un programa on l'usuari ingressi en un prompt la temperatura a Celcius i el programa mostri amb un **alert** la temperatura a Fahrenheit.
1.2 Demanar un número a un usuari a través d'un prompt i després dividir-lo per 10, exemple: 5/10 igual 0.5

2 If i else

2.1 Crear un programa que determini si un número introduït en un **popup** és divisible per 11 i 5 o no, mostrar el resultat amb **console.log**
2.2 Crear un programa que demani a l'usuari dos números en un Prompt i després mostri en un **alert** el nombre més gran.

3 While

3.1 Mostrar tots els números de 1 a **N** augmentant de 2 en 2 on **n** ho ingressa l'usuari des d'un prompt.

4 Patrons con Loop anidados

5 Array

Donat l'array = [1,2,3,4,5,6]
5.1 Iterar per tots els elements dins un array utilitzant `while` i mostrar-los a la pantalla.

5.2 Iterar per tots els elements dins un array utilitzant `for` i mostrar-los a la pantalla.

5.3 Iterar per tots els elements dins un array utilitzant .`for cada element` i mostrar-los a la pantalla.

5.4 Donat un array que conté `["blau", "groc", "vermell", "verd", "cafè", "rosa"]` determinar si un color introduït per l'usuari a través d'un prompt es troba dins de l'array o no.

6 Unió, intersecció i compteig

6.1 Hi ha dos arrays, cadascun amb 5 paraules, generar un nou array amb la intersecció dels dos elements. (Exemple: [1,2,3] unió [1,2,4] = [1,2])

6.2 Hi ha dos arrays, cadascun amb 5 paraules, generar un nou array amb la unió d'ambdós elements, (Exemple: [1,2,3] unió [1,2,4] = [1,1,2,2,3,4])

7 Arrays d'arrays

7.1 Crear una funció que permet juntar un array donat.

7.2 Crear una funció que divideixi un array en dues parts amb la mateixa quantitat d'elements (o una diferència de màxim un element) i torni un array que contingui el grup1 i el grup2.
